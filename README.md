## Requirements

`Toggler` is written in Swift 3. Compatible with iOS 8.0+

## Installation

### Cocoapods

Toggler is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Toggler'
```
### Carthage
```
github "younatics/Toggler"
```
